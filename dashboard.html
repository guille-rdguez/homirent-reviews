Links Generales
Cómo Usarlos
Formulario general: úsalo para compartir por mensaje o cuando recepción no necesite fijar un complejo.
Link iPad: deja el complejo bloqueado y activa auto-reset para operación continua en recepción.
Link QR o WhatsApp: mantiene la propiedad preseleccionada, pero cambia el `source` para medir mejor el origen de cada review.
