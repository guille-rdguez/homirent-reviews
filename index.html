HOMI

Reviews Local Launcher

Corre el proyecto en local sin build.

Esta portada sirve como punto de entrada local para abrir el formulario de huéspedes o el dashboard interno desde el mismo servidor estático.

Publico

Formulario

Versión pensada para iPad, WhatsApp o QR. Guarda reviews directamente en Supabase.

Abrir `form.html`
Interno

Dashboard

Panel para monitorear reviews, filtrar por periodo y revisar analytics por complejo.

Abrir `dashboard.html`
Inicia el servidor con `python3 serve-local.py` y luego entra a `http://127.0.0.1:4173/`.
